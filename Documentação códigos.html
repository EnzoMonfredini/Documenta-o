Documentação do Código - Sistema de Controle de Dados

Introdução

Este documento descreve o funcionamento e a estrutura do código do Sistema de Controle de Dados, que é desenvolvido em Python. O sistema utiliza várias bibliotecas, como gspread, mysql.connector, tkinter, openpyxl, para gerenciar e automatizar a leitura de planilhas, cálculo de estatísticas e inserção de dados em um banco de dados MySQL.

Visão Geral

O Sistema de Controle de Dados tem como objetivo permitir aos usuários acessar e analisar planilhas, calcular diversas estatísticas relacionadas aos dados contidos nelas e armazenar essas estatísticas em um banco de dados MySQL. As principais funcionalidades incluem:

  • Acesso à Planilha: O sistema permite ao usuário acessar uma planilha no Google Sheets com base no mês fornecido.
  • Cálculo de Estatísticas: Calcula várias estatísticas relacionadas às informações da planilha, incluindo contagem de infrutíferas, contagem de origens, categorização de programação e rota extra, contagem de NO SHOW e muito mais.
  • Inserção no Banco de Dados: As estatísticas calculadas podem ser inseridas em um banco de dados MySQL para análises futuras.
  • Exportação para Excel: As estatísticas podem ser exportadas para uma planilha Excel para fins de relatórios ou arquivamento.

Bibliotecas Principais

O código faz uso das seguintes bibliotecas Python:

  • gspread: Utilizado para autenticação e manipulação de planilhas do Google Sheets.
  • mysql.connector: Usado para a conexão e manipulação de um banco de dados MySQL.
  • tkinter: Para criar a interface gráfica do aplicativo.
  • openpyxl: Para criar e salvar planilhas Excel.
  • google.oauth2: Para autenticação com o arquivo JSON.
  • datetime: Para lidar com datas e horários.

Utilização

O sistema requer que o usuário forneça o mês para acessar a planilha. Após a leitura da planilha, as estatísticas podem ser calculadas e inseridas em um banco de dados MySQL ou exportadas para uma planilha Excel. A interface gráfica facilita o uso e interação com o sistema.

Conclusão

Este código é uma demonstração de como usar Python e várias bibliotecas para automatizar tarefas relacionadas à leitura e análise de dados em planilhas. Ele oferece funcionalidades de acesso a planilhas, cálculo de estatísticas e armazenamento em um banco de dados para análises futuras.

Documentação do Código - Sistema de Transportes

Introdução

Este documento descreve o funcionamento e a estrutura do código de um sistema de transportes desenvolvido em Python. O sistema utiliza bibliotecas como gspread, pandas, pyperclip, tkinter, e outras, para gerenciar e automatizar a comunicação com planilhas do Google Sheets, criar mensagens de atendimento e realizar ações relacionadas ao transporte de cargas.

Visão Geral

O sistema é composto por duas principais funcionalidades:

  • Envio de Mensagens: Permite ao usuário criar mensagens de atendimento para motoristas com base em informações de uma planilha do Google Sheets. A mensagem é criada, copiada para a área de transferência e, opcionalmente, enviada via WhatsApp.
  • Relatório de Carga e Descarga: Permite ao usuário visualizar e pesquisar informações de uma planilha do Google Sheets relacionadas ao transporte, bem como gerar um relatório de carga e descarga.

Bibliotecas Principais

O código faz uso das seguintes bibliotecas Python:

  • gspread: Para autenticação e manipulação de planilhas do Google Sheets.
  • pandas: Para processamento e manipulação de dados tabulares.
  • pyperclip: Para copiar mensagens para a área de transferência.
  • tkinter: Para criar a interface gráfica do aplicativo.
  • google.oauth2: Para autenticação via arquivo JSON.
  • pywhatkit: Para enviar mensagens via WhatsApp.
  • pandastable: Para exibir dados em tabelas na interface gráfica.

Funções Principais

O código é dividido em várias funções que desempenham papéis específicos. Aqui estão as funções mais relevantes:

  • login(): Realiza a autenticação no Google Sheets usando credenciais do arquivo JSON.
  • leitor(nome_planilha, nome_aba): Lê os dados de uma planilha e aba específicas no Google Sheets e os converte em um DataFrame do Pandas.
  • criar_mensagem(): Cria mensagens de atendimento com base nas informações da planilha e permite copiá-las para a área de transferência.
  • enviar_mensagem(): Envia a mensagem criada via WhatsApp, se o número de telefone do destinatário for encontrado.
  • visualizar_planilha(): Permite ao usuário visualizar a planilha do Google Sheets na interface gráfica.
  • pesquisar_na_planilha(): Permite ao usuário pesquisar informações na planilha com base em um termo de pesquisa.
  • gerar_relatorio_carga_descarga(): Gera um relatório de carga e descarga, que pode ser implementado posteriormente.

Interface Gráfica

O código utiliza a biblioteca tkinter para criar uma interface gráfica com duas abas: "Envio de Mensagens" e "Relatório de Carga e Descarga". O usuário pode interagir com o sistema por meio dessa interface.

Utilização

O sistema requer a inserção de informações como o nome do mês, nome do atendente e ID da carga para criar mensagens de atendimento. O usuário pode escolher entre o "Dia Atual" e o "Dia Seguinte" para buscar informações na planilha. A função de pesquisa só está disponível após a visualização inicial da planilha.

Conclusão

Este código é uma demonstração de como usar Python e bibliotecas de terceiros para automatizar tarefas relacionadas ao transporte e atendimento. Ele oferece funcionalidades de leitura de planilhas, criação de mensagens e interação com serviços de mensagens instantâneas.

Para informações adicionais e personalizações, consulte o código-fonte e adapte-o às suas necessidades específicas.
